\textbf{1. Решаемая проектом проблема и ее значимость} \\
Проект относится к проблеме анализа информации, оставленной пользователями в Интернете о лекарственных препаратах и медицинской сфере в целом.	 


\textbf{2. Поставленные в проекте цели и задачи} \\
Цель проекта состоит в применении современных средств прикладной математики для анализа данных.

Основные задачи проекта: 
\begin{itemize} 
\item Определение тональности отзывов методами машинного обучения 
\item Многоклассовая классификация отзывов 
\item Подбор гиперпараметров для эффективной реализации методов 
\item Визуализация данных на платформе Tableau Public
\end{itemize}
 

\textbf{3. Насколько новы и работоспособны представленные методики и решения}\\
Для выполнения задач использованы следующие методы: 
\begin{itemize} 
\item Для определения тональности:
	\begin{itemize} 
		\item Метод логистической регрессии
	\end {itemize}

\item Для многоклассовой классификации: 
	\begin{itemize} 
		\item Random Forest
		\item Метод опорных векторов (SVM)
		\item Градиентный бустинг над решающими деревьями
	\end {itemize}
\end{itemize} 


\textbf{4. Персональный вклад каждого члена проектной команды}\\
Получены следующие результаты:
\begin{itemize}
\item Визуализация данных на платформе Tableau Public (Горчавкина А.А.) 
\item Разработана модель машинного обучения по определению тональности отзывов (Пехтерев Д.О.)
\item Реализованы три метода многоклассовой классификации в среде Python: 
	\begin{itemize} 
		\item Random Forest
		\item Метод опорных векторов (SVM)
		\item Градиентный бустинг над решающими деревьями (Чертенков В.И.)
	\end {itemize} 
\end{itemize} 

\textbf{5. Демонстрируемый результат проекта}\\ 
	\begin{itemize} 
		\item Программный алгоритм, определяющий тональность текста, в основе которого лежит логистическая регрессия
		\item Программный алгоритм, классифицирующий тексты по соответствующим группам отзывов
		\item Визуализация в виде дашборда на платформе Tableau Public метрик качества тренировки модели машинного обучения 
	\end {itemize} 
	
\textbf{6. Апробация результатов}\\ 
	Использованные методы справились с задачей с достаточной точностью.
	
\textbf{7. Потенциал применения и внедрения}\\ 
Данный учебный проект имеет перспективы коммерческого применения. Итоговой продукт может заинтересовать как производителей лекарственных средств, так и площадок, предоставляющих аналитику для пользовательского потребления.
